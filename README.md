# Agrippa — De occulta philosophia: 1510 vs 1533

A scholarly comparison site mapping every chapter of Agrippa von Nettesheim's *De occulta philosophia libri tres* across its two major redactions: the Würzburg manuscript draft (W, ca. 1510) and the first printed Cologne edition (K, 1533).

**Live site:** https://t3dy.github.io/AgrippaDOP/

---

## What This Is

Heinrich Cornelius Agrippa completed an early draft of his three-book compendium of occult philosophy around 1510 and sent it in manuscript to the abbot Johannes Trithemius. The text circulated in this form for over twenty years before Agrippa revised and published it in Cologne in 1533. The two versions differ substantially: dozens of chapters were added wholesale, others were merged, reordered, or substantially expanded.

This site presents a card-based index of all 203 chapters and 4 epistles — one card per chapter, colour-coded by how the 1533 text relates to the 1510 draft:

| Badge | Meaning |
|-------|---------|
| **New in 1533** | Chapter has no 1510 counterpart |
| **Heavily Expanded** | 1510 material plus many additions |
| **Revised & Expanded** | 1510 material with shorter additions |
| **Preserved from W** | Essentially identical in both versions |
| **Largely New** | Only a small portion derives from 1510 |

Each card links to an essay page giving the Latin title, the Perrone Compagni collation note, and a scholarly discussion of the chapter's place in the revision.

The correspondence data is drawn entirely from the critical apparatus of:

> V. Perrone Compagni (ed.), *De occulta philosophia libri tres*, Leiden: Brill, 1992. (Studies in the History of Christian Traditions, vol. 48)

No continuous text from the Brill edition is reproduced. All extracts are short identification strings (e.g. "W, 1: 2 + two short additions") sourced from the edition's Table of Comparison. English chapter titles are taken from Donald Tyson's 1993 Llewellyn translation, which is in wide public use.

## Pipeline Documentation

The site is generated by a six-phase Python pipeline that reads the Perrone Compagni PDF and produces structured JSON consumed by the site builder. **The PDF itself is not included** (it is copyright Brill, 1992) and is excluded by `.gitignore`.

### Phase 1 — PDF Inspection (`scripts/01_inspect_pdf.py`)

Reads the edition PDF with [PyMuPDF](https://pymupdf.readthedocs.io/) (fitz) and produces a structural inventory:

- Total page count, embedded-text coverage percentage
- Identification of the manuscript siglum (confirmed: **W** = Würzburg MS M.ch.q.50)
- Page ranges for the Table of Comparison, sigla list, and critical apparatus
- Output: `outputs/reports/pdf_inspection.md`

The Perrone Compagni edition was digitised with ABBYY FineReader 14, yielding approximately 97% embedded-text coverage across 672 pages.

### Phase 2 — Text Extraction (`scripts/02_extract_pages.py`)

Extracts every page's text into a newline-delimited JSON file (`data/ocr/pages.jsonl`). Each record:

```json
{"page_number": 64, "text": "TABLE OF COMPARISON\n1:1\nW, 1: 1\n..."}
```

The raw extraction is stored locally only and excluded from the repository.

### Phase 3 — Structure Detection (`scripts/03_detect_structure.py`)

Classifies each page by content type: `edited_text`, `edited_text+apparatus`, `critical_apparatus`, `book_chapter_heading`, `comparison_table`, `front_matter`, `index`, etc. Results are written to `data/indices/edition_structure.json` with fields:

```json
{
  "page_number": 64,
  "section_type": "comparison_table",
  "heading": "TABLE OF COMPARISON",
  "confidence": "HIGH"
}
```

Navigation hints (comparison table page range, apparatus page list) are stored as top-level fields and consumed by later phases.

### Phase 10 — Comparison Table Parsing (`scripts/10_parse_comparison_table.py`)

This is the most technically involved phase. The Table of Comparison (pp. 64–69) maps each 1533 chapter to its 1510 W counterpart. The OCR layout varies by page:

**Row-by-row mode (p. 64, chapters 1–15):** Chapter reference and description appear on alternating lines:
```
1:1
W, 1: 1
1:2
W, 1: 2 + two short additions
```

**Column-by-column mode (pp. 65–69):** The two-column table was OCR'd left-column-first, then right-column:
```
1:37
1:38
...
[all refs for this page]
W, 1: 37 + ...
W, 1: 38 + ...
...
[all descriptions]
```

Additionally, on page 64 (chapters 16–36), ABBYY split compound tokens across lines:
- `1: 16` → `1\n16`  
- `W, 1:9` → `W, 1\n9`

The parser handles all three modes:
1. A `reconstruct()` pre-pass merges line-split artefacts
2. Mode detection examines the first 8 substantive lines; if all are chapter refs with no descriptions → column-by-column mode
3. A `recover_fragmented_p64()` pass uses a multi-line regex to recover the 21 split entries on page 64

Output: `outputs/tables/comparison_table.json` — 203 entries covering 4 Epistles, 74 Book I chapters, 60 Book II chapters, and 65 Book III chapters.

### Phase 11 — Chapter Title Extraction (`scripts/11_extract_chapter_titles.py`)

Latin chapter titles are extracted from the edited-text pages. The extractor:

1. Builds a page → book map using `heading` fields in `edition_structure.json` (detecting "LIBER SECUNDUS"/"TERTIUS"/"PRIMUS" transitions)
2. Scans all edited-text lines for `Cap. [roman numeral]` tokens
3. For each match, collects title text from the preceding 1–3 lines, stopping at structural noise (page numbers, book headings, index headings)

Output: `outputs/tables/chapter_titles.json` — 155 Latin titles (60 Book I, 43 Book II, 52 Book III). Chapters without a recoverable Latin title fall back to a slug in the essay page.

### Phase 12 — Site Generation (`scripts/12_build_dop_site.py`)

Reads `comparison_table.json` and `chapter_titles.json` and generates the complete static site:

- `site/index.html`: card grid with search input and status-filter dropdown; grouped into Epistles, Book I, Book II, Book III sections with chapter counts and status breakdowns
- `site/style.css`: parchment (`#f5f0e8`) and burnt sienna (`#8b4513`) design system
- `site/app.js`: client-side filter (text search + badge-class filter)
- `site/chapters/<slug>.html`: one essay page per chapter, containing:
  - English title (from embedded dictionary of 204 titles)
  - Latin title (from `chapter_titles.json` where available)
  - Status badge
  - Perrone Compagni collation note
  - 3-paragraph scholarly essay on the chapter's place in the revision

## Running the Pipeline Locally

You will need:
- Python 3.10+
- PyMuPDF: `pip install pymupdf`
- The Perrone Compagni Brill 1992 PDF (not included; copyright Brill)

```bash
# Place the PDF at:
data/raw/de_occulta_philosophia_perrone_compagni_1992.pdf

# Run phases in order:
python scripts/01_inspect_pdf.py
python scripts/02_extract_pages.py
python scripts/03_detect_structure.py
python scripts/10_parse_comparison_table.py
python scripts/11_extract_chapter_titles.py
python scripts/12_build_dop_site.py

# Site is built at site/
# Open site/index.html in a browser, or serve locally:
python -m http.server 8080 --directory site
```

The `outputs/tables/` JSON files (comparison table and chapter titles) are included in this repository so the site can be rebuilt without the PDF.

## Copyright and Fair Use

This project reproduces **no continuous text** from the Perrone Compagni critical edition. What is included:

- Short collation strings from the Table of Comparison (e.g. "W, 1: 2 + two short additions") — these are editorial apparatus identifying manuscript correspondences, not literary text
- Latin chapter headings extracted from the edition (these headings derive from the 16th-century manuscript, not from Perrone Compagni's editorial work)
- English chapter titles from the 1993 Tyson translation

All other content (essays, descriptions, summaries) is original commentary.

The `data/` directory is listed in `.gitignore`; no OCR'd text from the edition is committed.
